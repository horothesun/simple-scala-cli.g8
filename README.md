Giter8 template to generate a simple Scala-CLI project.

## Requirements

- Scala-CLI [install](https://scala-cli.virtuslab.org/install).

## Generate new project

```bash
scala-cli new horothesun/simple-scala-cli.g8
cd <repo_name>
```
